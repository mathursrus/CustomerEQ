# Component Catalog — Annex Cloud

## Navigation

### Primary Navigation
- **Type**: Sticky top nav, hamburger-menu style (mobile-first)
- **Logo**: Triangle gradient logo (red/pink/orange) + "ANNEX CLOUD" wordmark
- **Desktop**: Logo left, hamburger icon right
- **Behavior**: Mega-menu dropdown on desktop; collapses on mobile
- **Background**: White with subtle shadow

### Footer Navigation
- **Layout**: 4-column grid: Brand/contact | Platform links | Use Cases | Resources/About
- **Footer includes**: Address (McLean VA + LA + India), phone numbers, social icons (LinkedIn, YouTube), newsletter signup
- **Legal links**: Privacy policy, Terms of use, Modern Slavery Act, SLA and support

---

## Buttons

### Primary CTA Button
- **Style**: Filled purple background, white text, rounded corners, right-arrow icon (→)
- **Examples**: "Request a demo", "Get platform highlights", "See it in action"
- **Hover**: Darker purple

### Secondary CTA Button
- **Style**: White background, dark border, rounded corners
- **Examples**: "Decline" (cookie modal)

### Ghost/Link Button
- **Style**: Text only with underline or colored link
- **Examples**: "Learn more", "View all resources"

### Carousel Controls
- **Style**: "← Previous" / "Next →" text buttons
- **State**: Previous disabled on first slide

---

## Cards

### Feature Card (3-column grid)
- **Structure**: Icon (figure) + H3/H4 heading + paragraph description
- **Background**: White or light gray
- **Used on**: Capabilities overview, platform page sections

### Testimonial Card
- **Structure**: Company logo (figure) + blockquote paragraph + author name (bold)
- **Background**: White with subtle border
- **Used on**: Platform overview page

### Resource Card
- **Structure**: Hero image + Category label (pill) + H3 title + "Read more" link
- **Types**: Blog, Infographic, Assessment
- **Used on**: Homepage resources section, platform page

### Use Case Card
- **Structure**: Icon + H3 title + short description + "Learn more" CTA
- **Displayed as**: Filterable grid
- **Used on**: /use-cases/

### Stats Bar
- **Structure**: 3 stats side by side: Icon + large number + label
- **Examples**: "$1B Brands use Annex Cloud", "125 Integrations", "80 Countries"
- **Background**: Dark navy
- **Used on**: Homepage hero section

---

## Forms

### Demo Request Form
- **Page**: /request-demo/
- **Fields**: First Name, Last Name, Work Email, Company Name, Phone, Company Size, Message (textarea)
- **CTA Button**: "Submit" (purple)
- **Trust signals**: Partner logos (SAP, Microsoft, Salesforce, Shopify, Adobe)
- **Privacy note**: Privacy policy link

### Newsletter Subscription Form (Footer)
- **Fields**: Work Email (single input)
- **CTA**: "Subscribe Now" (purple button)
- **Privacy note**: "By completing this form, you agree to Annex Cloud's Privacy Policy"

### Cookie Consent Modal
- **Style**: White card overlay (bottom center)
- **Content**: Cookie usage explanation + Privacy Policy link
- **Buttons**: "Accept" (purple filled) + "Decline" (white outlined)

---

## Carousels / Sliders

### Hero Carousel (Homepage)
- **Type**: Auto-rotating tab carousel
- **Controls**: Numbered dot tabs (1, 2, 3)
- **Content**: Heading + paragraph + CTA per slide
- **Behavior**: Auto-advances, click to jump

### Feature Carousel
- **Type**: Horizontal scroll with Previous/Next arrows
- **Used on**: "Deliver rewarding experience" section, "What's new" section
- **Shows**: 3 cards visible at once

### Testimonial Carousel
- **Type**: Horizontal scroll with Previous/Next arrows
- **Shows**: 1 testimonial at a time
- **Content**: Logo + quote + author

---

## Layout Patterns

### Hero Section
- **Structure**: Left: H1 + paragraph + CTA | Right: Hero image
- **Background**: White
- **Image**: Photography (people using devices)

### Two-Column Feature Section
- **Structure**: Left: Image | Right: H2 + paragraph + feature list
- Alternates image side between sections

### Dark Section (Navy Background)
- **Background**: #1a1a3e dark navy
- **Text**: White
- **Used for**: Stats bars, CTAs, "Loyalty is all we do" section

### Full-Width CTA Banner
- **Background**: Purple gradient
- **Content**: H2 + "Request a demo" button
- **Used at**: Bottom of most pages

---

## Typography

- **Heading font**: Sans-serif, bold weight (appears to be custom/brand font)
- **Body font**: Clean sans-serif
- **H1**: Large, dark navy, bold
- **H2**: Medium-large, dark navy, bold
- **H3/H4**: Section headings, slightly smaller
- **Body**: Gray, regular weight, 16px base
- **Links**: Purple color, underline on hover
