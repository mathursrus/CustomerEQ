🤝 Shared structure

Both Send paths render the same React page component (<DistributePage mode="..."/>) parameterized by URL ?mode=. The full audience builder and the Common fields card are identical across modes; the operator can switch modes mid-flow without losing their audience or common-field values.

  • Shared: Distribution entry-point tile (two peer buttons); page header + "Switch mode" link; Audience builder (Existing-Members wildcard search + Random sample + Custom List paste/CSV + auto-enroll; unified deduped list with per-row deselect + 25/50 pagination); Common fields (Survey name in mail + Links expire on); Survey-detail Sent-count line; Distribution-batches mode pill.

🔀 Mode-specific divergence

Only the composer / generator section and the terminal-state surface diverge — three distinct subcomponents on each side of the split.

  • SELF_SERVE diverges: composer = format dropdown only · CTA = Generate N links · terminal = Success banner + Download CSV (preserved from #378).
  • MANAGED_ACS diverges: composer = sender + subject + rich-text body with mustache palette + live email preview · CTA = Send N emails · terminal = Sending → Sent SSE progress with per-recipient table + Retry-failed.
